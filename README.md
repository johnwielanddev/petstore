Pet Store API
=============

A very simple pet store API. Only /store/ is implemented.

Setup:
$ mkvirtualenv petstore
$ pip3 install -r requirements.txt

Run:

$ workon petstore
$ python3 app.py

Test
----

All tests:
$ python3 -m unittest

Integration tests:
$ python3 -m unittest test_integration

Unit tests:
$ python3 -m unittest test_unit
